## 📝 Enunciado

11 - Desenvolva um programa em Java que leia os valores de **A**, **B** e **C** de uma  
equação do segundo grau e mostre o valor de **Delta**.  

Fórmula:  

\[
\Delta = b^2 - 4ac
\]

---

## 🚀 Funcionalidades

- Leitura dos valores **A**, **B** e **C**  
- Cálculo do **Delta (Δ)**  
- Exibição do resultado no console
